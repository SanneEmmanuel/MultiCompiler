# Sanne MultiCompiler

Sanne MultiCompiler is a sleek, Visual Studio Code–style web IDE that supports multi-language compilation including C, C++, Go, V, and Carbon.  
It features a modern UI with tabbed editors, a hidden terminal that pops up with compilation results, and uses online compiler APIs to execute your code seamlessly.

---

## Features

- **Multi-language support:** Write and compile code in C, C++, Go, V, and Carbon.
- **VS Code–style interface:** Sidebar with icons, top toolbar, tabbed editor views, and a hidden terminal panel.
- **Tabbed editing:** Easily switch between multiple languages with persistent code per tab.
- **Hidden terminal:** Shows compile/run results with a simple toggle on compilation.
- **API-powered compilation:** Integrates with Go Playground API and Judge0 for compiling and running code.
- **Keyboard shortcut:** Press `Ctrl+Enter` (or `Cmd+Enter` on Mac) to quickly compile your code.
- **Responsive design:** Clean and modern UI styled with CSS for a smooth developer experience.
- **Open source:** Fully open source and easy to extend.

---

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/SanneEmmanuel/MultiCompiler.git
````

2. Open the project folder.

3. Open the main IDE interface in your browser:

   ```
   Src/index.html
   ```

4. Start coding, switch languages, and compile your code with the toolbar or keyboard shortcut.
